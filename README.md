# Super Small 'Font
**_Super Small 'Font_** is, as its name suggests, a tiny soundfont, at only **15 KB** of disk space taken. It makes use of only 6 samples and 6 instruments mapped across the 128 GM-compatible presets. GS-compatible drumsets aren't supported (only the standard drumset is available, and it doesn't support any GS-compatible sounds either).

This soundfont is perfect if you're running out of space, especially if you have an old PC from the 1990s. (But who still uses those in the first place? 😅)

Samples are taken from the Microsoft GS Wavetable Synth (thanks to William Borges for ripping them), except they're greatly shortened and crushed to a freq rate of 11025 Hz.
